# ATP5F1B_project# ATP5F1B Phylogenetic Analysis

## Data
- Raw sequences collected from UniProt
- File: data/raw/atp5f1b_raw.fasta

## Methods
- Alignment performed using MAFFT
- Phylogenetic tree constructed using Neighbor-Joining method

## Results
- Alignment: data/aligned/atp5f1b_alignment.fasta
- Tree: results/tree_sequence.nwk

## Reproducibility
All steps can be reproduced using the provided data and tools.
